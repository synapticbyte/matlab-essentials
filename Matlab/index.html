üîß MATLAB Beginner's Cheatsheet

Essential commands and functions for electrical engineering projects

üöÄ Basic Commands

clc
clc
Clears the command window
>> clc % Screen becomes clean
clear
clear / clear all / clear varname
Removes variables from workspace
>> clear all % Removes all variables
>> clear x % Removes only variable x
help
help function_name
Shows help documentation for any function
>> help sin % Shows help for sine function
who / whos
who / whos
Lists variables in workspace (whos shows details)
>> who % Lists variable names
>> whos % Shows size, class, etc.
format
format short/long/bank
Changes display format of numbers
>> format long % Shows more decimal places
>> format short % Default format
disp
disp(variable/text)
Displays variable value or text
>> disp('Hello World')
>> disp(x)

üìä Variables & Data Types

Variable Assignment
variable_name = value
Creates and assigns values to variables
>> x = 5
>> name = 'John'
>> pi_val = 3.14159
Arrays
[a b c] or [a, b, c]
Creates row vectors (arrays)
>> v = [1 2 3 4 5]
>> grades = [85, 90, 78, 92]
Column Vectors
[a; b; c]
Creates column vectors
>> col = [1; 2; 3; 4]
>> temps = [25; 30; 28; 32]
Ranges
start:step:end or start:end
Creates sequences of numbers
>> x = 1:10 % 1,2,3...10
>> t = 0:0.1:5 % 0,0.1,0.2...5
>> y = 10:-1:1 % 10,9,8...1
length
length(array)
Returns number of elements in array
>> x = [1 2 3 4]
>> length(x) % Returns 4
size
size(matrix)
Returns dimensions of matrix
>> A = [1 2; 3 4]
>> size(A) % Returns [2 2]

üßÆ Mathematical Operations

Basic Operations
+ - * / ^
Addition, subtraction, multiplication, division, power
>> 5 + 3 % 8
>> 10 - 4 % 6
>> 2 * 6 % 12
>> 8 / 2 % 4
>> 2^3 % 8
Element-wise Operations
.* ./ .^
Operations on each element of arrays
>> x = [1 2 3]
>> y = [4 5 6]
>> x .* y % [4 10 18]
Trigonometric
sin, cos, tan, asin, acos, atan
Trigonometric functions (angles in radians)
>> sin(pi/2) % 1
>> cos(0) % 1
>> tan(pi/4) % 1
Exponential & Log
exp, log, log10, sqrt
Exponential, natural log, log base 10, square root
>> exp(1) % e ‚âà 2.718
>> log(10) % Natural log
>> sqrt(16) % 4
Statistical
mean, median, std, max, min, sum
Statistical functions for data analysis
>> x = [1 2 3 4 5]
>> mean(x) % 3
>> max(x) % 5
>> sum(x) % 15
abs, round, ceil, floor
abs(x), round(x), ceil(x), floor(x)
Absolute value, rounding functions
>> abs(-5) % 5
>> round(3.7) % 4
>> ceil(3.2) % 4
>> floor(3.8) % 3

üìà Plotting & Visualization

plot
plot(x, y)
Creates 2D line plots
>> x = 0:0.1:10;
>> y = sin(x);
>> plot(x, y)
xlabel, ylabel, title
xlabel('text'), ylabel('text'), title('text')
Adds labels and title to plots
>> xlabel('Time (s)')
>> ylabel('Amplitude')
>> title('Sine Wave')
grid
grid on / grid off
Turns grid on/off in plots
>> plot(x, y)
>> grid on % Shows grid lines
hold
hold on / hold off
Keeps current plot when adding new plots
>> plot(x, sin(x))
>> hold on
>> plot(x, cos(x))
>> hold off
legend
legend('label1', 'label2')
Adds legend to distinguish multiple plots
>> plot(x, sin(x), x, cos(x))
>> legend('sin(x)', 'cos(x)')
subplot
subplot(rows, cols, position)
Creates multiple plots in one figure
>> subplot(2,1,1); plot(x,sin(x))
>> subplot(2,1,2); plot(x,cos(x))
stem, bar, histogram
stem(x,y), bar(x,y), histogram(data)
Different plot types for various data
>> stem(1:5, [2 4 1 5 3]) % Discrete plot
>> bar([1 2 3], [10 20 15]) % Bar chart
axis, xlim, ylim
axis([xmin xmax ymin ymax]), xlim([min max])
Controls axis limits and scaling
>> plot(x, y)
>> axis([0 5 -2 2]) % Set limits
>> xlim([0 10]) % X-axis only

üî¢ Matrix Operations

Matrix Creation
[row1; row2; row3]
Creates matrices using semicolons for new rows
>> A = [1 2 3; 4 5 6; 7 8 9]
>> B = [1 0; 0 1] % 2x2 identity
Matrix Indexing
A(row, col)
Access specific elements or submatrices
>> A(2,3) % Element in row 2, col 3
>> A(1,:) % All elements in row 1
>> A(:,2) % All elements in col 2

üíª Programming Constructs

for loop
for i = start:end ... end
Repeats commands for specified iterations
for i = 1:5
fprintf('i = %d\n', i)
end
while loop
while condition ... end
Repeats while condition is true
x = 1
while x < 10
x = x * 2
disp(x)
end
if statement
if condition ... elseif ... else ... end
Conditional execution of code
if x > 0
disp('Positive')
elseif x < 0
disp('Negative')
else
disp('Zero')
end
function
function output = funcname(input)
Creates custom functions (save as .m file)
function y = square(x)
y = x^2;
end

% Usage: result = square(5)
Comparison Operators
==, ~=, <, >, <=, >=
Compare values (returns true/false)
>> 5 == 5 % true (1)
>> 3 ~= 4 % true (1)
>> 2 < 3 % true (1)
>> 5 >= 5 % true (1)
Logical Operators
&& (AND), || (OR), ~ (NOT)
Combine logical conditions
>> (5>3) && (2<4) % true
>> (5>7) || (2<4) % true
>> ~(5>3) % false
fprintf
fprintf('format', variables)
Formatted output to command window
>> x = 3.14159
>> fprintf('Pi = %.2f\n', x)
% Output: Pi = 3.14
input
variable = input('prompt')
Gets user input from command window
>> name = input('Enter name: ', 's')
>> age = input('Enter age: ')

üìÅ File Operations

save
save filename variables
Saves variables to .mat file
>> x = [1 2 3]; y = [4 5 6]
>> save mydata x y
>> save('results.mat', 'x', 'y')
load
load filename
Loads variables from .mat file
>> load mydata
>> load('results.mat')
csvread/csvwrite
csvread('file.csv'), csvwrite('file.csv', data)
Read/write CSV files for data exchange
>> data = csvread('measurements.csv')
>> csvwrite('output.csv', results)
xlsread/xlswrite
xlsread('file.xlsx'), xlswrite('file.xlsx', data)
Read/write Excel files
>> [data, txt] = xlsread('data.xlsx')
>> xlswrite('results.xlsx', matrix, 'Sheet1')
fopen/fclose
fid = fopen('file.txt', 'r/w')
Open/close files for reading or writing
>> fid = fopen('data.txt', 'r')
>> data = fscanf(fid, '%f')
>> fclose(fid)
pwd, cd, ls
pwd, cd('path'), ls
Show current directory, change directory, list files
>> pwd % Shows current folder
>> cd('C:\MyProject') % Change folder
>> ls % List files in current folder
